# LangGraph Practice & Implementations

Welcome to my LangGraph learning repository! This project serves as a hands-on workspace for understanding and implementing AI agents using the [LangGraph](https://langchain-ai.github.io/langgraph/) framework.

## Repository Structure

*   **/Course**: This directory contains my heavily modified, hands-on implementations originating from the official [LangChain Academy - Introduction to LangGraph Course](https://github.com/langchain-ai/langchain-academy). I have converted the original architecture to securely utilize open-weights models via **ChatGroq (Llama-3.3)** instead of OpenAI, and fully integrated proper local LangSmith tracing. My sincere gratitude and attribution go to the LangChain Academy for the phenomenal source material.
*   **/Tutorials**: This section contains personalized code-alongs, standalone scripts, and deep-dives following the official LangGraph documentation.

## Core Technologies
*   **LangGraph & LangChain**: Core orchestration and agent framework.
*   **Groq (Llama-3.3)**: High-speed open-weights inference.
*   **LangSmith & LangGraph Studio**: State persistence, observability, and debugging.
